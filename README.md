## 📋 О Проекте

Этот проект представляет собой полностью рабочую стратегию алгоритмической торговли для криптовалютного рынка (BTC/USDT). Стратегия основана на комбинации индикаторов **RSI** и **SMA200** с динамическим управлением выходом из позиции.

### ✨ Ключевые особенности

- ✅ **Бэктест на реальных данных** — загрузка истории с Binance API
- ✅ **Трендовый фильтр** — вход только по направлению тренда (SMA200)
- ✅ **Умный выход** — фиксация прибыли при перегретом RSI
- ✅ **Гибкая конфигурация** — все параметры в файле `.env`
- ✅ **Подробное логирование** — каждая сделка записывается в лог
- ✅ **Оптимизировано под реальный рынок** — 5 итераций оптимизации
